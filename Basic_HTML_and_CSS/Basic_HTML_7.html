The details tag

A section of code that is not necessary for the user but can be toggled on and off by the user.

The summary tag defines a visible heading for the details element. The heading can be clicked to view/hide the details. - FROM w3schools.com

The time tag is used to produce smarter search results. It makes times machine readable. It's 1:00pm.

Also, Christmas recently passed. The datetime attribute gives more information about what day the day you are talking about is on.
